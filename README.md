Portfolio React — instructions rapides

Contenu:
- index.html
- src (App.jsx, main.jsx)
- assets (profile.svg, cv_badarou.pdf, style.css)
- package.json (scripts: start, build, deploy)

Installation locale:
1. Installer Node.js (16+)
2. npm install
3. npm run dev
4. npm run build -> crée le dossier dist

Déploiement GitHub Pages:
1. Créer repo: elbader630.github.io
2. Builder en local: npm run build
3. Copier le contenu du dossier 'dist' et l'uploader sur le dépôt (root)
   ou utiliser 'gh-pages' si tu préfères.

Dis-moi si tu veux que je te fournisse la commande Git exacte pour pousser directement dans ton repo.
